# Pintura JavaScript example project

This projects uses a test version of Pintura.

This version of Pintura will show a watermark in the editor and on generated images.

Purchase a license at https://pqina.nl/pintura to use Pintura in production.

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/pqina/pintura-example-javascript?title='Pintura JavaScript example project'&file=index.html)

## Installation

```bash
npm install
```

## Run

```bash
npm start
```
